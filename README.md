# Burnout Detection – AI-Powered Developer Burnout Detection

Burnout Detection analyzes GitHub commit patterns to predict developer burnout risk using machine learning. Built with Flask, MongoDB Atlas, and a modern 3D SaaS frontend.

## Features
- 🔐 JWT Authentication
- 📊 GitHub Integration (public commits)
- 🧠 Feature Engineering (night coding, sentiment, consistency, etc.)
- 🤖 ML Prediction (RandomForest with XGBoost/LSTM structure)
- 🎨 Futuristic UI with glassmorphism, neon gradients, and 3D effects
